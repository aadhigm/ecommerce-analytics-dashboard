# E-Commerce Sales & Customer Analytics Dashboard

## Project Overview
This project analyzes e-commerce sales data to identify trends, customer behavior, and business insights using Python, SQL, and Power BI.

## Tools Used
- Python (Pandas, NumPy)
- SQL
- Power BI

## Key Insights
- Sales show seasonal peaks during year-end
- Technology category generates highest revenue
- Western region performs best
- Top customers contribute significant revenue

## Project Structure
- data/ → dataset
- notebook/ → Python analysis
- dashboard/ → Power BI file
- sql/ → queries

## Dashboard Features
- KPI cards
- Sales trend
- Category & region analysis
- Customer insights
- Interactive filters
